## setup(作成中)
仮想環境を構築（python3のコマンドは実行できる前提）
```
python3 -m venv venv
```

仮想環境を有効化
```
source venv/bin/activate
```

各種インストール
```
venv/bin/python3 -m pip install --upgrade pip
pip3 install -r requirements.txt
```

main.pyを実行
```
python3 main.py
AtcoderのユーザーIDを入力
```

出てきた差分をPushしてマージ
